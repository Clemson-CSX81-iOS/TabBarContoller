- (IBAction) addView: (id)  sender

This method adds another view to the TabBarController. First we make sure our internal array is created, then create the new VC from the storyboard. Next we set the main label to the correct value and add the new ViewController to the TabBarController and allow moving the tabs.

- (IBAction) removeView: (id)  sender

This method removes the last ViewContoller from the TabBarController. This is done by getting the last object from the internal array then removing it from both arrays.
